Для запуска приложения необходимо из папки проекта перейти в server/ и запустить mvn spring-boot:run
Аналогичные действия необходимо произвести в web/

Также можно собрать jar через "mvn clean package" и запустить как "java -jar" (jar файлы приложены)

Сервер будет запущен на http://localhost:8081/
Клиент на http://localhost:8080/
Swagger на http://localhost:8081/swagger

Корректные данные для добавления нового пользователя:
Имя - любой текст
Логин - любой текст
Пароль - не менее 8-ми символов, с одной буквой маленькой, большой, цифрой и спец символом, типа '! (пример: qwertyQWERTY123'!)
Дата рождения - число.месяц.год (пример: 01.01.1990)
